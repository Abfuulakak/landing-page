    What we do

    Testing A Computer's Vision

    The project aimed to develop a Python based computer vision solution for image classification. The task was to create a model that could accurately classify images into predefined categories. Obtained a set of labelled images for training and testing (cat and dog) that were processed into 100 by 100 pixels and converted them into 1D arrays. Evaluated the model's performance by calculating accuracy on the tests on average. As a result, it achieved an accuracy of 87% on the test dataset demonstrating its effectiveness and accurately predicted the category of a new image as ‘cat’ showing the model's practical use. The project displayed competence in machine learning and image classification in Python. Though the project was a simplified version it provided foundations for more complex vision tasks in future.
